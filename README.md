# 📱 习概刷题神器 (云端适配版 v3.0)

这是一个专为**移动端刷题**设计的 Web 应用。支持单选、多选、判断题，具备智能纠错和自动解析功能。
**v3.0 更新**：完美适配云服务器部署，修复了题库格式不规范（粘连、全角标点）导致的识别错误。

![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg) ![Streamlit](https://img.shields.io/badge/Streamlit-Mobile%20Optimized-ff4b4b.svg)

## ✨ 核心亮点

*   **📱 手机端原生体验**：大字体、大按钮，单手操作无压力。
*   **🧠 智能容错解析**：
    *   自动分离粘连的题目和选项（如 `1.题目A.选项`）。
    *   自动兼容全角/半角标点（如 `.` 和 `．`）。
    *   判断题自动识别中文“对/错”并转换为选项。
*   **☁️ 云端部署友好**：支持 `0.0.0.0` 监听，轻松部署到腾讯云/阿里云。

## 🛠️ 快速部署

### 1. 安装依赖
```bash
pip install streamlit
```

### 2. 准备题库
新建 `tiku.txt`，将题库内容完整粘贴进去（支持单选、多选、判断）。

### 3. 启动服务 (云服务器)
为了让外网能访问，请使用以下命令启动：
```bash
# 前台运行测试
streamlit run web_quiz.py --server.address=0.0.0.0

# 后台长期运行 (推荐)
nohup streamlit run web_quiz.py --server.address=0.0.0.0 --server.port=8501 > quiz.log 2>&1 &
```

### 4. 访问
打开手机浏览器，输入 `http://<你的公网IP>:8501` 即可。

> **注意**：请务必在云服务器的安全组/防火墙中放行 TCP **8501** 端口！

## 📝 题库格式示例

系统支持极强的容错性，标准格式如下：

```text
一、单项选择题
1. 题目内容...
A. 选项A
B. 选项B
答案：A
答案解析：...

三、判断题
20. 题目内容...
答案：对
答案解析：...
```

*(简答题会被自动过滤)*

## ❓ 常见问题

**Q: 为什么手机打不开？**
A: 
1. 检查云服务器安全组是否开放了 8501 端口。
2. 检查启动命令是否加了 `--server.address=0.0.0.0`。

**Q: 如何停止后台运行？**
A: 输入 `ps -ef | grep streamlit` 找到进程 ID，然后 `kill -9 <进程ID>`。

**Q: 修改了题库没反应？**
A: 点击网页右上角的“Rerun”或者刷新浏览器即可。
